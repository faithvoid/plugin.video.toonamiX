# ToonamiX
Toonami Aftermath add-on for XBMC4Xbox. Backported from the Kodi "TATV" addon.

![](icon.png)

![1](screenshots/1.png)

## How To Use:
- Download & extract the latest release file
- Copy the "ToonamiX" folder to Q:/plugins/video
- Select your timezone and enjoy!
